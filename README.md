# Nixtune

~Note: Still Under Development~

Nixtune will recommend tunings for your Linux server based on various
softare that you run.

## Usage

Run the following to get started.

    python nixtune.py
    
## Recommendations

 - [X] Networking
 - [ ] FS
 - [X] IO
 - [X] Memory
